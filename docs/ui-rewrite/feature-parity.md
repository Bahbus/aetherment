## Notes for rewrite scoping
- Option rendering is schema-driven (`meta.options`), so parity should preserve dynamic rendering behavior rather than hardcoding individual option controls.
- “Preset import/export” currently exists only as clipboard payload exchange (base64-encoded JSON), not file-based preset I/O.
- “Import Mods” supports multi-select and async install with shared progress infrastructure.
